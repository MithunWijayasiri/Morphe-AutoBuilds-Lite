What you get

Patch. Sign. Ship.

Every day, we download stock APKs, apply community patches, sign them, and publish — so you don't have to.

01

Rolling release, always fresh

Every day the pipeline checks for updates. New versions get patched and uploaded to the same release — no manual work, no stale APKs.

$ check_app_updates.py
→ youtube: new version
→ instagram: unchanged
→ tiktok: new version
2 apps queued for rebuild
02

Three architectures, one keystore

arm64-v8a, armeabi-v7a, and universal builds. All signed with the same key — update in place, no reinstall.

arm64-v8a armeabi-v7a universal
03

Precise patch control per app

Each app gets exactly the patches it needs through declarative config. Empty patch list = apply all. Specific list = cherry-pick.

# patches/youtube-morphe.txt
# empty → apply all

# patches/tiktok-icysymmetra.txt
remove-ads
disable-login-wall
